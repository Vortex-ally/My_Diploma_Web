## 📖 About

Volunteer Management System is a web application designed to simplify volunteer coordination.

The system allows administrators to create and manage volunteer tasks, while volunteers can register, view available opportunities, and participate in events.

The project is actively being improved to demonstrate modern Django development practices.

## ✨ Features

- 👤 User authentication
- 🙋 Volunteer registration
- 📋 Task management
- 📅 Event management
- 🛡️ Django Admin panel
- 📱 Responsive interface
- 🔒 Secure authentication
